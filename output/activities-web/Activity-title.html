    Linear and Quadratic Functions as Shifts and Stretches.

    The families of all linear and quadratic functions can each be built using simpler functions: \(l(x) = mx\) (for positive values of \(m\)) and \(q(x) = ax^2\) (for positive values of \(a\)), for linear and quadratic functions, respectively. What does this mean exactly? It means every linear function is a shift and/or reflection of \(l(x)\) and every quadratic function is a shift and/or reflection of \(q(x)\text{.}\) In this worksheet we’ll explore this idea.
    🔗

    1.

    Consider the problem: find a linear function \(y = f(x)\) with slope \(2\) whose \(y\)-intercept is at \(y=7\text{.}\)
    🔗
    (a)
    Find the formula for \(f(x)\) and sketch a graph of it.
    🔗
    🔗
    🔗
    (b)
    The function \(f(x)\) you found is a vertical shift of a simpler function. Which simpler function is it? Explain in your own words and show this simpler function in your graph.
    🔗
    🔗
    (c)
    Now suppose that instead of having a \(y\)-intercept at \(y=7\text{,}\) we want a function that passes through the point \((3,7)\text{.}\) What additional transformation needs to be applied to your basic function from part (b)? Find the formula for this transformation and graph it. How else have we described this formula for a linear function?
    🔗
    🔗
    🔗

    3.

    Another way to find the vertex of a quadratic using simpler functions and shifts. Let’s see it through an example. Consider the function
    \begin{equation*} g(x) = 2x^2-6x+20. \end{equation*}
    🔗
    We can write this as a vertical shift of the function \(f(x) = 2x^2-6x\) upward by \(20\) units. Notice that
    \begin{align*} f(x) \amp = 2x^2 - 6x\\ \ \amp = 2x(x-3) \end{align*}
    🔗
    Notice that \(f(x)\) (from which \(g(x)\) came) has easily identified zeros at \(x=0\) and \(x=3\text{.}\) It’s vertex lies halfway between them at \(x=1.5\text{.}\) Then we can compute the \(y\)-value of this vertex to be
    \begin{equation*} f(1.5) = 2\cdot 1.5 \cdot (-1.5) = -4.5. \end{equation*}
    Thus, the vertex of \(g(x)\) is \(20\) units higher at \(y = 15.5\text{.}\) So the vertex form of \(g(x)\) is
    \begin{equation*} g(x) = 2x^2-6x+20 = 2(x-1.5)^2 + 15.5 \end{equation*}
    Your task: Use the idea from the above example to put the function
    \begin{equation*} h(x) = -3x^2+5x - 7 = -3x\left(x-\frac{5}{3}\right) -7 \end{equation*}
    into vertex form.
